Pabile Keyboards are custom-designed small keyboards and macropads. This repository contains the source files for our keyboard projects, including prototypes, designs, and configuration files. Please note that these files are intended as backups and may still be a work in progress. [Wiki](https://github.com/pabile/Pabile-Keyboards/wiki) [FAQ](https://sites.google.com/site/pabile/faq)
